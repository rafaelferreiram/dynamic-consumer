# Dynamic Consumer

Kafka consumer application that will consume data produced by the [dynamic](https://github.com/rafaelferreiram/dynamic) based on topics sent by the users and save them into ElasticSearch.

## Technologies
- Springboot
- Java 11
- Kafka
- ElasticSearch
